Common questions

Why does the math just round?

Stitch counts must be whole numbers. Some patterns require specific multiples (e.g., for stitch repeats); the rounded result is a starting point — round again to your nearest pattern multiple if needed.

Will you add cm/inch toggle?

v1 uses 4-inch (10 cm) gauge convention. Explicit unit toggle is on the v1.1 list.

Multiple project counters?

v1 has one shared counter (for the project you're working on right now). Per-project counters are planned for a later update.
